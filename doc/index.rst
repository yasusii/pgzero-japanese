Pygame Zero へようこそ
======================

Pygame Zero はお決まりの手続きをあれこれ書かなくても、ゲームを作れるようにしたものです。

教育の場での利用を意図して作られており、先生がPygame API やイベントループの説明をしなくても、基本的なプログラミングを教えられるようになっています。
