Pygame Zero へようこそ
======================

Pygame Zero を使うと決まりきった手続きをいちいち書かかずにゲームを作成できます。

Pygame Zero は教育の場での利用を意図しており、指導者は Pygame の API やイベントループの説明をしなくても、基本的なプログラミングを教えることが可能になります。
